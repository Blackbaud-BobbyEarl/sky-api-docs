Acknowledgement

The acknowledgement entity has the following properties:

Property Type Description
status string The status of the acknowledgement. Available values are acknowledged, notacknowledged, and donotacknowledge.
date fuzzy date The date associated with the acknowledgement.

 

Gift

The gift entity has the following properties:

Property Type Description
id string The gift identifier.
date dateTime The gift date. Uses ISO-8601 format: 1969-11-21T10:29:43-04:00.
date_added dateTime The date the gift was added. Uses ISO-8601 format: 1969-11-21T10:29:43-04:00.
date_modified dateTime The date the gift was last modified. Uses ISO-8601 format: 1969-11-21T10:29:43-04:00.
amount currency The monetary amount of the gift.
type string The gift type. Available values are listed below.
constituent_id string The constituent identifier.
is_anonymous boolean Indicates whether the gift is anonymous.
gift_splits array of gift split The set of gift splits associated with the gift.
linked_gifts array of string The identifiers of other gifts that are linked to this gift.
subtype string The subtype of the gift. Available values are the entries in the Gift Subtypes table.
lookup_id string The gift lookup identifier.
balance currency The balance remaining on the gift.
acknowledgement acknowledgement The acknowledgement associated with the gift.
receipt receipt The receipt of the gift.
soft_credits array of soft credit The set of soft credits associated with the gift.
fundraisers array of fundraiser The set of fundraisers who should receive credit for this gift.
post_status string The general ledger post status of the gift. Available values are posted, notposted, and donotpost.
post_date fuzzy date The date that the gift was posted to general ledger.
gift_status string The status of the gift. Available values are active, held, terminated, completed, and cancelled.
gift_aid_qualification_status string The gift aid qualification status of the gift. Available values are qualified, notqualified, and partlyqualified. For UK only.
gift_aid_amount currency The amount of tax reclaimed from gift aid for this gift. For UK only.
reference string Notes to track special details about a gift such as the motivation behind it or a detailed description of a gift-in-kind.
payment_method string The payment method of the gift. Available values are listed below.

 

Available values for the type of a gift:

Type Description
amendment An amendment.
donation A donation.
giftinkind A gift-in-kind.
generalledgerreversal A general ledger reversal.
matchinggiftpledge A matching gift pledge.
matchinggiftpayment A matching gift payment.
matchinggiftwriteoff A matching gift write off.
plannedgift A planned gift.
pledge A pledge.
pledgepayment A pledge payment.
pledgewriteoff A pledge write off.
recurringgift A recurring gift.
recurringgiftpayment A recurring gift payment.
stock A stock.
soldstock A sold stock.
other Another type of gift.

 

Available values for the payment method of a gift:

Type Description
cash Cash.
personalcheck A personal check.
businesscheck A business check.
creditcard A credit card.
standingorder A standing order.
directdebit Direct debit.
voucher A voucher.
other Another method of payment.

 

Gift attachment

The gift attachment entity has the following properties:

Property Type Description
id string The gift attachment identifier.
parent_id string The parent object's identifier.
type string The gift attachment type. Available values are link and physical. Physical gift attachments are stored and managed in the system. Link gift attachments are stored and managed externally. At this time, only link gift attachments can be created through the API.
url string The URL for the gift attachment. The URL for a physical gift attachment contains a time-bound signature that limits access to 60 minutes.
tags array of string The set of tags associated with the gift attachment. Available values are the entries in the Document Tags table.
name string The name of the gift attachment.
content_type string The content type. Only applies to physical gift attachments.
file_name string The file name. Only applies to physical gift attachments.
file_size long The file size in bytes. Only applies to physical gift attachments.
thumbnail_url string The URL for a thumbnail. Only applies to physical gift attachments that are images. Contains a time-bound signature that limits access to 60 minutes.

 

Gift fundraiser

The gift fundraiser entity has the following properties:

Property Type Description
amount currency The amount credited to the fundraiser for the gift.
constituent_id string The constituent identifier.

 

Gift split

A gift split is the distribution of the total amount of a gift among multiple campaigns, funds, or appeals.

The gift split entity has the following properties:

Property Type Description
fund_id string The identifier of the fund associated with the gift split.
amount currency The amount of the gift split.
campaign_id string The identifier of the campaign associated with the gift split.
appeal_id string The identifier of the appeal associated with the gift split.

 

Receipt

The receipt entity has the following properties:

Property Type Description
status string The status of the receipt. Available values are receipted, notreceipted, and donotreceipt.
date fuzzy date The date on the receipt.
number string The number of the receipt.
amount currency The amount of the receipt.

 

Soft credit

A soft credit is a way to share the credit for a gift with other individuals such as friends or spouses.

The soft credit entity has the following properties:

Property Type Description
amount currency The amount of the soft credit.
constituent_id string The constituent identifier.
id string The soft credit identifier.
gift_id string The gift identifier.
