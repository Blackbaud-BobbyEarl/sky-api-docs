AdjustmentCategory

The AdjustmentCategory entity has the following properties:

Property Type Description
adjustment_category_id integer The identifier for this adjustment category.
category string The category for this adjustment category.
type string The type of this adjustment category.
transfer_to_account_id integer The transfer account of this adjustment category.
amount number The amount of this adjustment category.

The AdjustmentDetail entity has the following properties:

Property Type Description
adjustment_id integer The identifier for this adjustment.

number string The number for this adjustment.
reference string The reference for this adjustment.

transaction_status string The transaction status for this adjustment.
post_status string The post status for this adjustment.
post_date date-time The post date for this adjustment.

notes string The notes for this adjustment.
distributions array of Distribution The distributions for this adjustment.

CustomField

CustomFieldOptions

The CustomFieldOptions entity has the following properties:

Property Type Description
custom_field_id integer The identifier for this custom field.
name string The name of this custom field.
type string The type (boolean, table, etc) of this custom field.
is_required boolean Whether this custom field is required.
table_values array of string List of table values. Only applicable if type = 'Table'.

Distribution

The Distribution entity has the following properties:

Property Type Description
distribution_id integer The distribution identifier.
amount number The distribution amount.
description string The description for this distribution
account_number string The account number for this distribution
type_code string The journal entry transaction type (debit or credit).
custom_fields array of CustomField The custom fields for this distribution.
distribution_splits array of DistributionSplit The distribution split

 

DistributionSplit

The DistributionSplit entity has the following properties:

Property Type Description
distribution_split_id integer The distribution split identifier.
ui_project_id string The user-visible identifier of the project.
transaction_code_values array of TransactionCodeValue The distribution transaction code values.
amount number The distribution journal entry amount. Only one of Amount or Percent can be specified.
percent number The distribution journal entry amount percentage. Only one of Amount or Percent can be specified.
account_class string The account class for this distribution split.
