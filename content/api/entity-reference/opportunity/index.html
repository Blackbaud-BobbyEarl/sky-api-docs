This topic describes the entity and type representations for common items that the Opportunity API uses. To learn about how to set up your developer account and work with the SKY API, see the Getting Started guide.

Campaign

The campaign entity has the following properties:

Property Type Description
id string The campaign identifier.
description string The campaign description.
lookup_id string The campaign's lookup identifier.

 

Collection

The collection entity has the following properties:

Property Type Description
count integer The total number of items in the collection.
next_link URI For paginated responses, this represents the URI for the next page of results.
value list Contains a collection of items included in this response (which may be a subset of the total items in the collection).

 

Currency

The currency entity has the following properties:

Property Type Description
value decimal The monetary value.

 

Fund

The fund entity has the following properties:

Property Type Description
id string The fund identifier.
description string The fund description.
lookup_id string The fund's lookup identifier.

 

Fundraiser

The fundraiser entity has the following properties:

Property Type Description
id string The opportunity fundraiser identifier.
constituent_id string The fundraiser identifier
credit_amount currency The amount the fundraiser is credited for the opportunity.

 

Opportunity

The opportunity entity has the following properties:

Property Type Description
id string The opportunity identifier.
constituent_id string The constituent identifier.
name string The opportunity name.
purpose string The purpose of the opportunity.
status string The status of the opportunity.
deadline dateTime The opportunity deadline.
ask_date dateTime The ask date.
ask_amount currency The ask amount.
expected_date dateTime The expected date.
funded_date dateTime The opportunity description.
funded_amunt currency The funded amount.
campaign campaign The campaign related to the opportunity.
fund fund The fund related to the opportunity.
fundraisers array of action fundraiser entities. The fundraisers assigned to the opportunity.
